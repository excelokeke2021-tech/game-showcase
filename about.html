About This Project

Welcome to my game showcase website. This project is built to display all the games I have, their details, and gameplay videos in one organized place.

🎮 What This Website Is

This is not a game store. It is a personal game library where I showcase all my installed games, upcoming games, and their details like story, gameplay, and videos.

⚙️ Features

  • Game details pages for each game
  • Gameplay video support (offline videos)
  • Search and filter system
  • Responsive design for mobile and PC
  • Organized game library structure

🧠 Purpose

The purpose of this site is to organize my games in a clean way, learn web development, and build a personal gaming showcase platform.

👤 Creator

Built by Excel — learning web development and creating a personal game library system.
